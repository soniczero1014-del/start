# 股票量化模拟网页设计稿

## 目标

制作一个简单的股票量化模拟及信息网页。第一版先不接真实行情，用本地示例数据，让页面可以直接打开、选择股票、查看信息、运行简单策略模拟，并看到结果。

## 页面结构

页面采用单页布局，分成三块：

1. 顶部标题区：说明这是股票量化模拟工具。
2. 股票信息区：展示股票名称、代码、价格、涨跌、成交量和一个简易走势条形图。
3. 量化模拟区：选择股票、输入初始资金、选择策略，然后运行模拟。
4. 模拟结果区：展示期末资产、收益率、初始资金、收益曲线和交易记录。

## 数据和策略

第一版内置 `AAPL`、`MSFT`、`TSLA` 三只示例股票。每只股票包含名称、价格、涨跌、成交量和一组历史收盘价。

策略先做两个：

1. 买入并持有：第一天尽量买入，最后一天计算资产。
2. 均线交叉：用短均线和长均线做简单买卖信号。

## 技术方案

用静态网页实现：

1. `index.html` 放页面结构。
2. `styles.css` 放页面样式。
3. `quant.mjs` 放示例数据和模拟计算逻辑。
4. `app.js` 负责把数据和结果显示到页面上。
5. `tests/quant.test.mjs` 用 Node 自带测试工具检查核心计算逻辑。

这样做的好处是第一版可以很快跑起来，后面也容易把示例数据换成真实行情接口。

## 验收标准

1. 浏览器能打开网页。
2. 页面能显示股票信息。
3. 切换股票后，股票信息和走势图会变化。
4. 输入初始资金并点击模拟后，会显示期末资产、收益率、收益曲线和交易记录。
5. 核心计算逻辑有自动测试。
